Time To Get Critical!...

First we need to define the X input.

This "X" is theorized to impact the "Y" of interest relative to the Fishbone, VOC, XY Matrix or pFMEA. It can be either a single X or a family of variations such as shift, operator, supplier, method, etc.

How is this X theorized to influence the Y?

This should be in the form of a business question which will drive the statistical question.

What tool or method will be used to measure the X?

What statistic will be tested and what statistical tool will be used to prove or disprove the business question?

Are we testing the Mean, Median or Variance? Maybe we are testing for correlation, doing a regression, etc.

State the Null Hypothesis.

Typically shows there is no difference between the sample sets tested or that there is no relationship seen.

State the Alternate Hypothesis.

Typically shows there is a difference between the sample sets tested or that there is a relationship seen.

What is the statistical result?

State the P- Value and/or other related pertinent stats.

What are the practical conclusions derived from doing this test?

What actions will you now take as a result of the findings? Do you plan on implementing a solution from this analysis? If not, say so.
